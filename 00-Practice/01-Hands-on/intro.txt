fs :
    fs ka matlab hai file system
    


fs.readdir(path[,options],callback)
        1.readdir -> iska use folder ke andar kya kya hai usko read karne ke liye hota hai
        2.path -> jaha se read karna hai us folder ka path
        3.options -> iska use folder ke andar kya kya hai usko read karne ke liye hota hai
        4.callback -> iska use folder ke andar kya kya hai usko read karne ke liye hota hai



res.render("index", { files });

        2.ejs -> iska use ejs file ko render karne ke liye hota hai

fs.writeFile(path,data[,options],callback)
    1.writeFile -> iska use file create karne ke liye hota hai
    2.path -> jaha se read karna hai us folder ka path
    3.data -> iska use file create karne ke liye hota hai
    4.options -> iska use file create karne ke liye hota hai
    5.callback -> iska use file create karne ke liye hota hai
